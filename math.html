Matrices

Matrix = Rectangular array of numbers
Order = m × n
Row Matrix → 1 × n
Column Matrix → m × 1
Square Matrix → n × n
Identity Matrix → diagonal = 1
(A+B)ij = aij + bij
(AB)ij = Σ aik bkj
AB ≠ BA
(AB)ᵀ = BᵀAᵀ
Aᵀ = A → symmetric
Aᵀ = −A → skew symmetric
AA⁻¹ = I
(AB)⁻¹ = B⁻¹A⁻¹
Idempotent → A² = A
Involutory → A² = I
Nilpotent → Aⁿ = 0

Determinants

|A| = determinant of matrix
Row interchange changes sign
If two rows equal → determinant = 0
Factor can be taken out
Minor → determinant after removing row & column
Cofactor Cij = (-1)^(i+j) Mij
adj(A) = transpose of cofactor matrix
A⁻¹ = adj(A) / |A|
