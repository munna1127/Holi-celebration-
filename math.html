1. Definition of Matrix

Matrix = Rectangular array of numbers arranged in rows and columns
Order of matrix = m × n (rows × columns)

2. Types of Matrices

Row Matrix → matrix having only one row
Column Matrix → matrix having only one column
Square Matrix → number of rows = number of columns
Zero Matrix → all elements are zero
Diagonal Matrix → non-zero elements only on diagonal
Scalar Matrix → diagonal elements equal
Identity Matrix → diagonal elements = 1
Upper Triangular Matrix → aᵢⱼ = 0 for i > j
Lower Triangular Matrix → aᵢⱼ = 0 for i < j

3. Equality of Matrices

Two matrices are equal if their corresponding elements are equal
A = B if aᵢⱼ = bᵢⱼ

4. Matrix Addition

Possible only when matrices have same order
(A + B)ᵢⱼ = aᵢⱼ + bᵢⱼ
Commutative property → A + B = B + A
Associative property → A + (B + C) = (A + B) + C

5. Scalar Multiplication

kA = multiply every element of matrix by scalar k

6. Matrix Multiplication

AB exists only if columns of A = rows of B
(AB)ᵢⱼ = Σ aᵢₖ bₖⱼ
Matrix multiplication is NOT commutative
AB ≠ BA
Associative property → A(BC) = (AB)C
Distributive property → A(B + C) = AB + AC

7. Transpose of Matrix

Transpose of matrix A is written as Aᵀ
(Aᵀ)ᵢⱼ = aⱼᵢ
(Aᵀ)ᵀ = A
(A + B)ᵀ = Aᵀ + Bᵀ
(kA)ᵀ = kAᵀ
(AB)ᵀ = BᵀAᵀ

8. Symmetric Matrix

A matrix is symmetric if
Aᵀ = A

9. Skew Symmetric Matrix

A matrix is skew symmetric if
Aᵀ = −A
Diagonal elements of skew symmetric matrix are always zero

10. Decomposition Theorem

Any square matrix can be written as sum of symmetric and skew symmetric matrices
A = ½(A + Aᵀ) + ½(A − Aᵀ)

11. Identity Matrix

AI = IA = A
I² = I

12. Inverse of Matrix

AA⁻¹ = I
A⁻¹A = I
Inverse exists only when |A| ≠ 0

13. Important Properties of Inverse

(AB)⁻¹ = B⁻¹A⁻¹
(Aᵀ)⁻¹ = (A⁻¹)ᵀ
(kA)⁻¹ = (1/k)A⁻¹

14. Solving Matrix Equations

AX = B → X = A⁻¹B
XA = B → X = BA⁻¹

15. Special Matrices

Idempotent Matrix → A² = A
Involutory Matrix → A² = I
Nilpotent Matrix → Aⁿ = 0

Chapter 2 – Determinants

1. Definition

Determinant is a special number associated with square matrix
Written as |A|

2. Determinant of 2×2 Matrix

| a b | | c d |
|A| = ad − bc

3. Determinant of 3×3 Matrix

| a b c | | d e f | | g h i |
|A| = a(ei − fh) − b(di − fg) + c(dh − eg)

4. Minor of Element

Minor = determinant obtained after deleting row and column of element

5. Cofactor

Cij = (−1)^(i+j) Mij

6. Adjoint of Matrix

adj(A) = transpose of cofactor matrix

7. Inverse using Determinant

A⁻¹ = adj(A) / |A|
Inverse exists only if |A| ≠ 0

8. Important Properties

If two rows are equal → determinant = 0
If two rows are proportional → determinant = 0
Row interchange changes sign of determinant
Common factor can be taken outside
If a row becomes zero → determinant = 0

9. Area of Triangle using Determinant

Area = ½ | x1 y1 1 x2 y2 1 x3 y3 1 |

10. Cramer's Rule

For equations a1x + b1y = c1 a2x + b2y = c2
x = Dx / D
y = Dy / D
