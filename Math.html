Chapter 1 – Matrices (Board Complete)

1. Definition

Matrix = Rectangular array of numbers
Order = m × n (rows × columns)

2. Types of Matrices

Row Matrix → 1 × n
Column Matrix → m × 1
Square Matrix → n × n
Zero Matrix → All elements 0
Diagonal Matrix → Non-zero only on diagonal
Scalar Matrix → Diagonal elements equal
Identity Matrix (I)
Upper Triangular → aᵢⱼ = 0 for i > j
Lower Triangular → aᵢⱼ = 0 for i < j

3. Equality of Matrices

A = B if aᵢⱼ = bᵢⱼ

4. Matrix Addition

Possible only if same order
(A + B)ᵢⱼ = aᵢⱼ + bᵢⱼ

5. Scalar Multiplication

kA = Multiply each element by k

6. Matrix Multiplication

AB possible if columns of A = rows of B
(AB)ᵢⱼ = Σ aᵢₖ bₖⱼ
Matrix multiplication NOT commutative
AB ≠ BA (in general)
Associative → A(BC) = (AB)C
Distributive → A(B + C) = AB + AC

7. Transpose of Matrix

Transpose of A = Aᵀ
(Aᵀ)ᵢⱼ = aⱼᵢ
(Aᵀ)ᵀ = A
(A + B)ᵀ = Aᵀ + Bᵀ
(AB)ᵀ = Bᵀ Aᵀ
(kA)ᵀ = kAᵀ

8. Symmetric Matrix

Aᵀ = A
Example form:
[ a b ]
[ b c ]

9. Skew Symmetric Matrix

Aᵀ = −A
Diagonal elements = 0
Example form:
[ 0 a ]
[ −a 0 ]

10. Decomposition Theorem

Any square matrix can be written as:
A = ½(A + Aᵀ) + ½(A − Aᵀ)
½(A + Aᵀ) → Symmetric
½(A − Aᵀ) → Skew Symmetric

11. Identity Matrix

AI = IA = A
I × I = I

12. Inverse Matrix

AA⁻¹ = I
A⁻¹A = I
Inverse exists only if |A| ≠ 0

13. Important Properties

(AB)⁻¹ = B⁻¹A⁻¹
(Aᵀ)⁻¹ = (A⁻¹)ᵀ
(kA)⁻¹ = (1/k)A⁻¹

14. Solving Matrix Equations

AX = B → X = A⁻¹B
XA = B → X = BA⁻¹

15. Finding Unknown Elements

If A = B then corresponding elements equal
a₁₁ = b₁₁
a₁₂ = b₁₂
a₂₁ = b₂₁
a₂₂ = b₂₂

16. Condition for Symmetric Matrix

Aᵀ = A
Compare elements and form equations

17. Condition for Skew Symmetric Matrix

Aᵀ = −A
Diagonal elements = 0

18. Matrix Commutativity Questions

AB = BA
Find unknown values using element comparison

19. Identity Matrix Trick

AI = IA = A
I² = I

20. Matrix Power

A⁰ = I
A¹ = A
A² = A × A

21. Board Favourite Pattern

If A² = A then A is idempotent matrix
If A² = I then A is involutory matrix
If Aᵀ = −A then A is skew symmetric

22. Special Matrix Types (Boards)

Idempotent → A² = A
Involutory → A² = I
Nilpotent → Aⁿ = 0

23. Board Repeated Questions (With Full Solution)

Q1. If A = B, find k

A =
[2 k]
[3 4]
B =
[2 5]
[3 4]
Solution:
If A = B then corresponding elements are equal
k = 5
Final Answer: k = 5
---

Q2. Find transpose of matrix

A =
[1 2 3]
[4 5 6]
Solution:
Transpose means rows become columns
Aᵀ =
[1 4]
[2 5]
[3 6]
Final Answer: Aᵀ =
[1 4]
[2 5]
[3 6]
---

Q3. Show that matrix is symmetric

A =
[2 3]
[3 4]
Solution:
Find transpose
Aᵀ =
[2 3]
[3 4]
Aᵀ = A
Therefore matrix is symmetric
---

Q4. Show matrix is skew symmetric

A =
[0 3]
[-3 0]
Solution:
Find transpose
Aᵀ =
[0 -3]
[3 0]
Aᵀ = −A
Therefore matrix is skew symmetric
---

Q5. Solve matrix equation AX = B

AX = B
Solution:
Multiply both sides by A⁻¹
A⁻¹AX = A⁻¹B
IX = A⁻¹B
X = A⁻¹B
